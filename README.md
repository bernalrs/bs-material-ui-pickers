## Installation

```

yarn add bs-material-ui-pickers
yarn add material-ui-pickers
```

Then add `bs-material-ui-pickers` to `bs-dev-dependencies` in your `bsconfig.json`:

```json
{
  ...
  "bs-dev-dependencies": ["bs-material-ui-pickers"]
}
```
